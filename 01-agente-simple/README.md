# Nivel 1: Agente Ultra Simple

Este es el nivel más básico del taller. Aquí aprendemos los fundamentos de strands:

## ¿Qué hace este agente?

- ✅ Crea un modelo LiteLLM básico
- ✅ Inicializa un agente con strands
- ✅ Mantiene una conversación simple
- ✅ No tiene herramientas (tools)
- ✅ No tiene planner-executor

## Cómo ejecutarlo:

```bash
cd 01-agente-simple
python agent.py
```

## Conceptos aprendidos:

- Importar strands
- Crear un modelo de lenguaje
- Inicializar un agente básico
- Hacer prompts al agente
